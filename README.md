# DevPortfolio AI 🚀

An intelligent full-stack SaaS application that automatically generates 
a professional developer portfolio website using GitHub API, PDF resume 
parsing, and Llama 3.3 AI (via Groq).

## 🌟 What it does
- Connect your GitHub → auto-imports all repositories, stars, languages
- Upload your PDF resume → extracts skills, experience, education
- AI generates your bio, project descriptions, and taglines
- Choose from 3 beautiful themes
- One-click publish → get a live shareable portfolio URL

## 🛠️ Built With
- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express.js  
- Database: MongoDB + Mongoose
- AI: Groq API (Llama 3.3 70B) — Free
- Auth: JWT + bcryptjs
- File Upload: Multer + pdf-parse
